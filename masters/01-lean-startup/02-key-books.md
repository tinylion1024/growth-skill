# 精益创业 - 必读书单

---

## 核心书籍

### 1. The Lean Startup

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   《精益创业》                                               │
│                                                             │
│   作者：Eric Ries                                          │
│   出版：2011                                                │
│   页数：336                                                 │
│   豆瓣：8.4                                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘

核心内容：

第一部分：愿景
   - 精益创业的起源
   - 精益创业的核心原则
   - 什么是MVP

第二部分：驾驭
   - Build-Measure-Learn循环
   - 验证性学习
   - 创新核算

第三部分：加速
   - 小批量
   - 增长引擎
   - 创新组织

精华摘录：

"精益创业的核心是：所有经营活动都是实验，
 用于验证对产品和市场的基础假设。"

"失败是学习的机会。快速失败，快速学习。"
```

### 2. Lean Analytics

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   《精益数据分析》                                           │
│                                                             │
│   作者：Alistair Croll / Benjamin Yoskovitz               │
│   出版：2013                                                │
│   页数：400                                                 │
│   豆瓣：8.2                                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘

核心内容：

六大发展阶段：
1. 冷启动 (Ecommerce → B2B)
2. 早期增长 (SaaS → 移动应用)
3. 规模化增长 (用户生成内容)
4. 营收增长 ( marketplace)
5. 成熟期 (品牌与忠诚度)
6. 衰退与转型

核心指标框架：
   - 单一关键指标 (One Metric That Matters)
   - 虚荣指标 vs 可执行指标
   - 精益画布

精华摘录：

"不要看你的用户怎么说，要看他们怎么做。"

"数据是洞察的起点，不是终点。"
```

## 阅读顺序建议

```
入门路径（按顺序）：

Step 1: 《精益创业》- 掌握核心思维
   → 理解MVP和BML循环

Step 2: 《精益数据分析》- 掌握指标框架
   → 学习如何用数据验证假设

Step 3: 《Running Lean》- 掌握实战工具
   → 学习精益画布和验证流程

Step 4: 实战应用
   → 将理论应用到实际项目
```
