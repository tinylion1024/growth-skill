# 定位与品牌体系

> Al Ries / Seth Godin | 为什么用户选择你

## 核心理念

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   定位决定增长                                                   │
│                                                                 │
│   传统思维：产品优先         定位思维：用户心智优先               │
│   ─────────────            ──────────────                      │
│   我的产品最好              用户认为谁最好就是谁最好              │
│   功能比对手强              在用户心智中占据位置                  │
│   酒香不怕巷子深            认知>事实                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

核心原则：
1. **市场是心智之战** - 用户的认知决定了选择
2. **位置 > 产品** - 在用户心智中占据位置比产品更重要
3. **Remarkable 才能传播** - 平庸的产品不会自传播
```

## 核心模型

### 定位理论 (Positioning)

```
定位基本框架：

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   定位 = 在用户心智中占据一个独特的位置                         │
│                                                                 │
│   定位三问：                                                     │
│   1. 你是谁？（品牌名）                                         │
│   2. 你是什么？（品类）                                          │
│   3. 为什么选你？（差异化）                                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

定位的六个步骤：

1. 分析市场
   - 市场有多大？
   - 主要玩家是谁？
   - 用户需求是什么？

2. 确定竞争对手
   - 用户当前用什么？
   - 替代方案是什么？

3. 选择差异化
   - 你与竞品的不同是什么？
   - 这个不同对用户重要吗？

4. 建立信任状
   - 如何证明你的差异化？
   - 有什么背书？

5. 传播定位
   - 用一句话说明你的定位
   - 一致性传播

6. 持续优化
   - 监控认知变化
   - 随市场调整
```

### Purple Cow（突出效应）

```
Seth Godin 的 Purple Cow 理论：

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   传统营销：找到目标消费者 → 打断他们 → 强迫他们看广告          │
│                                                                 │
│   Purple Cow：                                                    │
│   创造一个值得注意的产品 → 让消费者主动谈论                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

Remarkable（值得注意）的标准：

1. 必须 remarkable
   - 值得被谈论
   - 超出预期

2. 必须可视化
   - 能被看到
   - 能被分享

3. 必须可复制
   - 能大规模推广
   - 有规模效应

4. 必须有意义
   - 对用户有价值
   - 能解决真实问题

从 Purple Cow 到增长：
- 好的 Purple Cow = 话题 = 口碑 = 增长
- 坏的 Purple Cow = 噪音 = 被忽略
```

## 关键概念

### 1. 品类思维

```
品类 > 产品：

在用户心智中：
- 不是"可口可乐vs百事可乐"
- 而是"可乐vs橙汁vs水"

品类选择：

用户决策路径：
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   品类思考 ──▶ 品牌选择                                         │
│                                                                 │
│   "我想喝点什么？"                                              │
│       │                                                        │
│       ├── 碳酸饮料 ── 可口可乐                                 │
│       ├── 果汁 ────── 农夫山泉                                  │
│       └── 水 ─────── 农夫山泉                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

品类塑造：
- 创造新品类
- 占领新品类
- 领先新品类
```

### 2. 差异化

```
差异化三层次：

1. 产品差异化
   - 功能差异
   - 性能差异
   - 设计差异

2. 运营差异化
   - 价格差异
   - 渠道差异
   - 服务差异

3. 认知差异化
   - 品牌差异
   - 定位差异
   - 故事差异

最高级的差异化是认知差异化：
- 产品可以被复制
- 运营可以被模仿
- 认知很难被改变

差异化检验：
- 用户能说清楚你的不同吗？
- 用户愿意告诉朋友你的不同吗？
```

### 3. 品牌叙事

```
品牌故事框架：

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   故事 = 冲突 + 转折 + 启示                                     │
│                                                                 │
│   冲突：用户遇到什么问题？                                       │
│   转折：你如何解决这个问题？                                      │
│   启示：用户获得了什么价值？                                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

好品牌故事的要素：

1. 真实
   - 不是编造
   - 用户能验证

2. 相关
   - 与用户痛点相关
   - 用户能代入

3. 简单
   - 一句话说清楚
   - 容易记住

4. 情感
   - 引发情感共鸣
   - 让用户记住
```

## 实践方法论

### 定位设计流程

```
定位设计五步法：

Step 1: 市场分析
   - 确定目标市场
   - 分析市场规模
   - 了解用户痛点

Step 2: 竞品分析
   - 列出主要竞品
   - 分析竞品定位
   - 找到差异化空间

Step 3: 差异化定位
   - 找到独特优势
   - 确定核心信息
   - 建立信任状

Step 4: 定位验证
   - 用户测试
   - 调整优化
   - 确认认知

Step 5: 传播执行
   - 一致性传播
   - 持续投入
   - 监控效果
```

### 品牌信息架构

```
品牌信息层级：

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   愿景（Vision）                                                │
│      │                                                         │
│      ▼                                                         │
│   使命（Mission）                                               │
│      │                                                         │
│      ▼                                                         │
│   价值观（Values）                                              │
│      │                                                         │
│      ▼                                                         │
│   品牌承诺（Promise）                                           │
│      │                                                         │
│      ▼                                                         │
│   品牌主张（Proposition）                                       │
│      │                                                         │
│      ▼                                                         │
│   关键信息（Key Messages）                                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

一句核心定位：
- 谁是目标用户
- 解决什么问题
- 提供什么价值

例：
- "小红书：标记我的生活" - UGC种草社区
- "抖音：记录美好生活" - 短视频内容平台
```
