# 定位与品牌 - 定位实操

> 如何在用户心智中占据位置

---

## 定位的本质

```
定位 = 回答一个问题：

为什么用户应该选择你而不是竞争对手？

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   定位不是：                                                    │
│   - 不是给产品起名字                                            │
│   - 不是设计一个logo                                            │
│   - 不是写一句广告语                                            │
│                                                                 │
│   定位是：                                                      │
│   - 在用户心智中找到一个独特的位置                              │
│   - 这个位置竞争对手没有占据                                    │
│   - 这个位置对目标用户有意义                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 定位方法论

### 经典定位四步法

```
Step 1: 确定竞争对手

分析工具：心智地图
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│                    用户心智中的竞争格局                          │
│                                                                 │
│         可口可乐                                                │
│              │                                                  │
│              │                                                  │
│      百事可乐 ─────── 王老吉                                   │
│              │                                                  │
│              │                                                  │
│         农夫山泉                                                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

在哪个品类竞争？
- 在可乐品类：可口可乐 vs 百事可乐
- 在凉茶品类：王老吉 vs 加多宝
```

```
Step 2: 找到差异化优势

差异化来源：
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   产品差异化          服务差异化          形象差异化             │
│   ─────────          ─────────          ─────────             │
│   功能                速度               品牌个性                │
│   设计                可靠性             视觉识别                │
│   性能                专业性             故事                     │
│   价格                售后               价值观                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

差异化检验：
1. 这个差异化对用户重要吗？
2. 竞争对手没有这个差异化吗？
3. 我们能证明这个差异化吗？
```

```
Step 3: 找到支持点

支持点类型：
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   物理支持             行为支持             认知支持             │
│   ─────────            ─────────            ─────────            │
│   品质认证              用户数据             名人背书              │
│   技术专利              客户案例             专家认可              │
│   市场份额              服务承诺             行业地位              │
│   生产工艺              运营历史             历史传承              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

好的支持点：
- 可以被验证
- 用户能感知
- 竞争对手难以复制
```

```
Step 4: 传播定位

定位传播原则：
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   一致性：                                                       │
│   - 所有传播都传递同一信息                                       │
│   - 品牌名、slogan、视觉都围绕定位                               │
│                                                                 │
│   简洁性：                                                       │
│   - 一句话说清楚定位                                             │
│   - 用户能在3秒内理解                                            │
│                                                                 │
│   重复性：                                                       │
│   - 持续传播定位                                                │
│   - 不轻易改变                                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 定位类型

### 品类定位

```
成为品类代表：

当你能代表一个品类时，用户会优先想到你

案例：
- 百度 = 搜索
- 淘宝 = 电商
- 微信 = 社交

品类定位策略：
1. 创造新品类
   - 你是第一个进入用户心智的新品类
   - 例：Tesla = 电动汽车

2. 占据已有品类
   - 在成熟品类中找到差异化位置
   - 例：BMW = 驾驶乐趣

3. 重塑品类
   - 挑战传统品类定义
   - 例：Airbnb = 旅行住宿
```

### 差异化定位

```
差异化定位策略：

1. 领导者定位
   "我们是行业第一"
   适用于：市场领导者
   案例：国酒茅台

2. 比附定位
   "我们和XX一样好，但更便宜"
   适用于：挑战者
   案例：蒙牛比附伊利

3. 特性定位
   "我们专注于XX特性"
   适用于：细分市场
   案例：王老吉 = 怕上火

4. 使用场景定位
   "我们是XX场景的最佳选择"
   适用于：多场景产品
   案例：红牛 = 困了累了

5. 用户定位
   "我们是XX用户的选择"
   适用于：细分人群
   案例：茅台 = 商务宴请
```

## 品牌叙事

### 品牌故事框架

```
好的品牌故事结构：

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   [冲突]：用户遇到了什么问题？                                    │
│                                                                 │
│   [转折]：我们发现了什么/创造了什么？                            │
│                                                                 │
│   [启示]：用户因此获得了什么价值？                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

品牌故事示例：

星巴克：
冲突：咖啡只是咖啡，没有体验
转折：星巴克创造了"第三空间"概念
启示：咖啡不只是咖啡，是社交体验

Apple：
冲突：电脑只是工具，没有美感
转折：Apple创造了设计精美的产品
启示：科技也可以很美

Patagonia：
冲突：消费主义破坏环境
转折：Patagonia坚持环保使命
启示：消费也可以有责任感
```

### 品牌语言体系

```
品牌语言要素：

1. 品牌名
   - 容易记忆
   - 暗示价值
   - 独特性

2. Slogan
   - 一句话说清楚定位
   - 情感连接
   - 可持续使用

3. Tone of Voice
   - 专业/亲切/权威？
   - 一致性的表达方式

4. Visual Identity
   - Logo
   - 色彩
   - 视觉风格

品牌语言示例：

Apple：
- 简洁
- 自信
- 创新
- "Think Different"

Nike：
- 激励
- 运动
- 成就
- "Just Do It"

小红书：
- 年轻
- 真实
- 种草
- "标记我的生活"
```

## 定位测试与优化

### 定位验证方法

```
定位验证三问：

1. 用户能说出来吗？
   - 问用户：你知道这个品牌吗？它是什么？
   - 如果用户说不清楚，定位不清

2. 用户愿意分享吗？
   - 用户愿意向朋友推荐吗？
   - NPS > 50 说明定位有效

3. 用户选择你吗？
   - 在决策时用户选择你吗？
   - 市场数据验证选择

定位健康检查：
- 认知度：用户知道这个品牌吗？
- 认知内容：用户知道这个品牌是做什么的吗？
- 认知偏好：用户认为这个品牌有什么优势？
- 选择率：用户真正选择这个品牌的比例？
```
