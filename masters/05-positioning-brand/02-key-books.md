# 定位与品牌 - 必读书单

---

## 核心书籍

### 1. Positioning

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   《定位》                                                  │
│                                                             │
│   作者：Al Ries / Jack Trout                               │
│   出版：1981（20周年版2001）                                │
│   页数：250                                                 │
│   豆瓣：8.3                                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘

核心内容：

第一部分：定位基础
   - 定位的诞生
   - 定位的原理
   - 用户心智

第二部分：定位方法
   - 分析心智
   - 找到位置
   - 传播定位

第三部分：定位案例
   - 各行业定位案例
   - 成功与失败

精华摘录：

"定位不是从产品开始，而是从用户心智开始。"

"在传播过度的社会中，定位是最有力的武器。"
```

### 2. Purple Cow

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   《紫牛》                                                  │
│                                                             │
│   作者：Seth Godin                                          │
│   出版：2003                                                │
│   页数：160                                                 │
│   豆瓣：7.8                                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘

核心内容：

第一部分：为什么需要紫牛
   - 传统营销失效
   - 广告被忽略
   - 需要值得注意的产品

第二部分：什么是紫牛
   - Remarkable = 值得被谈论
   - 从人群中脱颖而出
   - 创造口碑

第三部分：如何创造紫牛
   - 设计值得注意的产品
   - 让用户成为传播者
   - 规模化传播

精华摘录：

"平庸的产品不会自传播。"

"紫牛是创造口碑的最佳方式。"
```

### 3. Obviously Awesome

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   《显然的完美》                                             │
│                                                             │
│   作者：April Dunford                                        │
│   出版：2019                                                │
│   页数：240                                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘

核心内容：

第一部分：定位基础
   - 产品市场匹配
   - 定位的重要性

第二部分：定位流程
   - 确定正确市场
   - 分析竞争格局
   - 确定差异化价值
   - 验证定位假设

第三部分：定位执行
   - 传递定位信息
   - 调整产品功能
   - 优化定价

精华摘录：

"定位的目标是让用户立刻理解你的价值。"

"最好的定位是用户自己说出来的。"
```

## 阅读顺序建议

```
入门路径（按顺序）：

Step 1: 《定位》- 基础理论
   → 理解定位的本质

Step 2: 《紫牛》- 产品设计
   → 理解如何创造值得注意的产品

Step 3: 《显然的完美》- 实操方法
   → 学习如何做定位

Step 4: 实战应用
   → 将理论应用到品牌建设
```
