## 核心书籍

### 1. Lean Analytics

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   《精益数据分析》                                           │
│                                                             │
│   作者：Alistair Croll / Benjamin Yoskovitz               │
│   出版：2013                                                │
│   页数：400                                                 │
│   豆瓣：8.2                                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘

核心内容：

第一部分：精益数据分析
   - 数据驱动决策
   - 精益分析框架
   - One Metric That Matters

第二部分：六阶段增长
   - 冷启动阶段
   - 早期增长阶段
   - 规模化增长
   - 营收增长
   - 成熟期
   - 衰退与转型

第三部分：行业指标
   - 电子商务指标
   - SaaS指标
   - 用户生成内容
   - marketplace

精华摘录：

"数据是洞察的起点，不是终点。"

"不要看你的用户怎么说，要看他们怎么做。"
```

### 2. Subscribed

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   《订阅》                                                  │
│                                                             │
│   作者：Tien Tzuo                                           │
│   出版：2018                                                │
│   页数：240                                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘

核心内容：

第一部分：订阅经济
   - 订阅的本质
   - 订阅 vs 一次性销售
   - 订阅的优势

第二部分：订阅增长
   - 获取订阅者
   - 激活订阅者
   - 留存订阅者
   - 扩张收入

第三部分：订阅运营
   - 客户成功
   - 续费与流失
   - 订阅指标

精华摘录：

"订阅是最古老的商业模式，也是最新的增长引擎。"

"在订阅经济中，留存比获取更重要。"
```
