# 增长学习路线图

> 6个月精通增长

---

## 学习目标

```
6个月后，你将掌握：
- 精益创业方法论
- 产品驱动增长
- 增长飞轮设计
- 商业模型分析
- 品牌定位战略
- 实战增长技能
```

---

## 学习路径

### Month 1: 精益创业（实验方法）

```
学习目标：
- 理解MVP思维
- 掌握Build-Measure-Learn循环
- 学会设计增长实验

核心任务：
□ 阅读《精益创业》
□ 完成精益画布
□ 设计一个MVP实验
□ 分析一个创业公司的精益方法

里程碑：
- 能用精益思维分析产品问题
- 完成一个增长实验设计
```

### Month 2: 产品驱动增长

```
学习目标：
- 理解PLG框架
- 掌握用户激活方法
- 学会留存分析

核心任务：
□ 阅读《产品驱动增长》
□ 分析产品的激活漏斗
□ 优化产品的Onboarding
□ 制定留存提升计划

里程碑：
- 能诊断激活率问题
- 设计有效的Onboarding流程
- 建立留存监控体系
```

### Month 3: 增长飞轮

```
学习目标：
- 理解四种增长引擎
- 掌握Growth Loop设计
- 学会渠道优化

核心任务：
□ 阅读《增长黑客》
□ 识别产品的增长引擎类型
□ 设计一个Growth Loop
□ 优化一个核心渠道

里程碑：
- 能分析产品的增长引擎
- 设计可持续的Growth Loop
- 优化渠道效率
```

### Month 4: 商业模型

```
学习目标：
- 掌握SaaS指标体系
- 理解单位经济模型
- 学会数据驱动决策

核心任务：
□ 阅读《精益数据分析》
□ 建立指标监控仪表盘
□ 分析单位经济模型
□ 优化LTV/CAC

里程碑：
- 能读懂SaaS指标
- 建立增长仪表盘
- 优化单位经济
```

### Month 5: 定位与品牌

```
学习目标：
- 理解定位理论
- 掌握品牌叙事
- 学会差异化竞争

核心任务：
□ 阅读《定位》
□ 为产品做定位分析
□ 设计品牌故事
□ 建立品牌信息体系

里程碑：
- 能分析竞品定位
- 为产品找到差异化定位
- 建立品牌叙事
```

### Month 6: 综合实战

```
学习目标：
- 整合所有增长知识
- 完成增长体系设计
- 实战项目复盘

核心任务：
□ 整理个人增长体系
□ 完成一个增长项目
□ 复盘与优化
□ 制定下一个季度计划

里程碑：
- 构建个人增长框架
- 完成至少一个增长实验
- 形成持续学习习惯
```

## 每月学习清单

### Month 1 详细计划

```
Week 1: 精益创业基础
- 阅读《精益创业》第1-5章

- 完成精益画布模板

Week 2: MVP设计
- 阅读《精益创业》第6-10章
- 理解MVP的不同类型
- 设计一个MVP方案

Week 3: Build-Measure-Learn
- 阅读《精益创业》第11-15章
- 掌握BML循环
- 设计一个实验

Week 4: 实战应用
- 分析一个产品的精益方法
- 完成一个增长实验设计
- 复盘学习成果
```

## 学习建议

```
成功学习的关键：

1. 理论与实践结合
   - 每学习一个概念，动手实践
   - 用真实产品验证理论

2. 输出优先
   - 写学习笔记
   - 做案例分析
   - 分享给他人

3. 社群学习
   - 加入增长社群
   - 与他人讨论
   - 寻找学习伙伴

4. 持续迭代
   - 定期复盘
   - 调整学习计划
   - 保持好奇心

常见问题：

Q：6个月能学会吗？
A：取决于投入时间。每天2小时，6个月可以入门。每天4小时，6个月可以精通。

Q：没有基础能学吗？
A：可以。从《精益创业》开始，循序渐进。

Q：学到什么程度算掌握？
A：能用增长思维分析产品问题，能独立设计增长实验，能建立增长监控体系。
```
