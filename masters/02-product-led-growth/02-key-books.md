# 产品驱动增长 - 必读书单

---

## 核心书籍

### 1. Product-Led Growth

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   《产品驱动增长》                                           │
│                                                             │
│   作者：Wes Bush                                            │
│   出版：2019                                                │
│   页数：256                                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘

核心内容：

第一部分：PLG基础
   - 什么是产品驱动增长
   - PLG vs 销售驱动
   - PLG的商业模型

第二部分：产品设计
   - Freemium设计
   - 用户激活
   - 留存机制

第三部分：商业化
   - 定价策略
   - 转化优化
   - 扩张收入

精华摘录：

"最好的增长方式是让产品自己说话。"

"PLG不是免费策略，而是让产品成为主要增长引擎。"
```

### 2. Inspired

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   《启示录》                                                │
│                                                             │
│   作者：Marty Cagan                                         │
│   出版：2010（第二版2017）                                   │
│   页数：256                                                 │
│   豆瓣：9.0                                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘

核心内容：

第一部分：产品管理
   - 产品经理职责
   - 产品发现方法
   - 用户研究

第二部分：产品设计
   - 产品原则
   - MVP与原型
   - 产品验证

第三部分：产品团队
   - 产品团队组建
   - 产品领导力
   - 团队协作

精华摘录：

"产品经理的职责是发现可以服务用户的产品机会。"

"技术团队和产品团队必须紧密合作。"
```

### 3. Hooked

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   《上瘾》                                                  │
│                                                             │
│   作者：Nir Eyal                                            │
│   出版：2014                                                │
│   页数：256                                                 │
│   豆瓣：7.5                                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘

核心内容：

上瘾模型（Hook Model）：
   Trigger → Action → Reward → Investment

触发：
   - 外部触发（Push通知、邮件等）
   - 内部触发（情绪、习惯）

行动：
   - 简化行为
   - 提高动机
   - 触发+能力+动机

奖励：
   - 社会奖励（社交）
   - 自我奖励（完成感）
   - 多变奖励（惊喜）

投入：
   - 增加后续触发
   - 存储价值
   - 加载下一触发

精华摘录：

"让用户养成习惯是产品的终极目标。"
```

## 阅读顺序建议

```
入门路径（按顺序）：

Step 1: 《启示录》- 产品管理基础
   → 理解产品团队如何工作

Step 2: 《上瘾》- 习惯养成机制
   → 学习如何让用户形成习惯

Step 3: 《产品驱动增长》- 商业化
   → 理解Freemium和商业模型

Step 4: 实战应用
   → 将理论应用到产品优化
```
