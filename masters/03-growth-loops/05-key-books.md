# 增长飞轮 - 必读书单

---

## 核心书籍

### 1. Hacking Growth

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   《增长黑客》                                               │
│                                                             │
│   作者：Sean Ellis / Morgan Brown                           │
│   出版：2017                                                │
│   页数：320                                                 │
│   豆瓣：8.5                                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘

核心内容：

第一部分：方法论
   - 增长团队组建
   - 增长漏斗
   - 快节奏实验

第二部分：实战
   - 用户获取
   - 用户激活
   - 用户留存
   - 用户变现
   - 用户推荐

第三部分：案例
   - Facebook、Airbnb、Dropbox
   - 实战案例分析

精华摘录：

"增长黑客是一套系统，让产品、市场、工程紧密合作。"

"最好的增长方法是让产品自己说话。"
```

### 2. The Cold Start Problem

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   《冷启动问题》                                             │
│                                                             │
│   作者：Andrew Chen                                         │
│   出版：2021                                                │
│   页数：320                                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘

核心内容：

第一部分：冷启动问题
   - 什么是冷启动
   - 为什么网络效应产品难启动
   - 冷启动的六种模式

第二部分：突破冷启动
   - 単边启动策略
   - 双边启动策略
   - 病毒启动策略

第三部分：规模化
   - 跨越鸿沟
   - 规模化增长
   - 维护网络效应

精华摘录：

"网络效应产品的核心挑战是：需要用户才能创造价值，
 但没有用户就没有价值。"

"冷启动的解决方案是找到那个'魔法数字'——让网络
 效应开始自运转的关键用户数。"
```

## 阅读顺序建议

```
入门路径（按顺序）：

Step 1: 《增长黑客》- 增长框架
   → 理解增长团队和漏斗

Step 2: 《冷启动问题》- 网络效应
   → 理解网络效应产品增长

Step 3: 《Traction》- 渠道选择
   → 找到适合你的增长渠道

Step 4: 实战应用
   → 将理论应用到实际增长
```
