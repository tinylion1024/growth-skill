# 增长飞轮 - 付费引擎

> LTV > CAC | 让增长可持续

---

## 付费引擎原理

```
付费增长公式：

 LTV > CAC

只要用户终身价值 > 获客成本
就能通过付费持续增长

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   付费增长飞轮：                                                 │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │                                                         │   │
│   │   投入广告 ────▶ 获取用户 ────▶ 产生收入                │   │
│   │       ▲              │              │                  │   │
│   │       │              │              │                  │   │
│   │       └──────────────┴──────────────┘                  │   │
│   │                                                         │   │
│   │                   收入 > 成本                          │   │
│   │                                                         │   │
│   └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│   关键指标：                                                    │
│   - LTV（用户终身价值）> CAC（获客成本）                        │
│   - Payback Period（回收周期）< 12个月                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 核心指标体系

### LTV（用户终身价值）

```
LTV 定义：

LTV = ARPU × 留存时长 × 毛利率

LTV = 平均每用户收入 × 用户生命周期 × 毛利率

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   LTV 计算示例：                                                 │
│                                                                 │
│   假设：                                                         │
│   - 月ARPU = 100元                                              │
│   - 平均留存 = 12个月                                            │
│   - 毛利率 = 70%                                                │
│                                                                 │
│   LTV = 100 × 12 × 70% = 840元                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

LTV 优化方向：
- 提高ARPU（定价、附加销售）
- 延长留存（减少流失）
- 提高毛利（优化成本）
```

### CAC（用户获取成本）

```
CAC 定义：

CAC = 总获客成本 / 新增用户数

CAC = 营销支出 + 销售支出 + 人员成本
      ─────────────────────────────────
              新增付费用户数

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   CAC 构成：                                                     │
│                                                                 │
│   直接成本：                                                     │
│   - 广告支出（Google、Facebook等）                               │
│   - 渠道返点                                                     │
│   - 促销优惠                                                     │
│                                                                 │
│   间接成本：                                                     │
│   - 销售团队人员成本                                             │
│   - 市场团队人员成本                                             │
│   - 工具和系统成本                                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

CAC 优化方向：
- 优化渠道效率
- 提高转化率
- 降低单位成本
```

### LTV/CAC 比率

```
LTV/CAC 比率：

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   比率           含义                    行动                    │
│   ───────────────────────────────────────────────────────────   │
│   LTV/CAC < 1    亏损                    停止投放                │
│   LTV/CAC 1-3    平衡                    优化效率                │
│   LTV/CAC 3-5    健康                    可以扩张                │
│   LTV/CAC > 5    优秀                    大力扩张                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

理想目标：LTV/CAC > 3
这是可持续增长的安全边界
```

### Payback Period（回收周期）

```
Payback Period 定义：

回收周期 = CAC / (ARPU × 毛利率)

回收周期越短，资金效率越高

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   回收周期参考：                                                  │
│                                                                 │
│   < 6个月   优秀   - 快速回收，资金效率高                         │
│   6-12个月  良好   - 可接受，持续投入                             │
│   12-18个月 一般   - 需要优化                                    │
│   > 18个月  危险   - 资金压力大，需要转型                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 付费渠道策略

### 主要付费渠道

```
付费渠道分类：

1. 搜索引擎（SEM/SEO）
   - Google Ads
   - 百度推广
   - 适合：明确需求，主动搜索

2. 社交媒体
   - Facebook/Instagram
   - 抖音/快手
   - 适合：品牌曝光，兴趣触达

3. 展示广告
   - 程序化购买
   - 适合：品牌曝光，引流网站

4. 视频广告
   - YouTube
   - 抖音视频
   - 适合：品牌+转化

5. 口碑/推荐
   - 联盟营销
   - 合作推广
   - 适合：低成本获客
```

### 渠道选择框架

```
渠道选择决策：

Step 1: 确定目标用户
   - 人口统计（年龄、性别、收入）
   - 行为特征（兴趣、习惯）
   - 购买阶段（认知、考虑、决策）

Step 2: 匹配渠道
   - 目标用户在哪个渠道？
   - 渠道触达成本？
   - 渠道转化效率？

Step 3: 测试和优化
   - 小规模测试
   - 测量ROI
   - 扩大投入

渠道效率矩阵：

              高转化率         低转化率
        ┌────────────────┬────────────────┐
   高   │   优先投入      │    测试优化     │
   覆   │                │                │
   盖   ├────────────────┼────────────────┤
   率   │                │                │
   低   │    扩张测试     │    避免投入     │
   覆   │                │                │
   盖   └────────────────┴────────────────┘
```

### 付费广告优化

```
广告优化框架：

1. 创意优化
   - 标题吸引力
   - 图片/视频质量
   - CTA清晰度
   - A/B测试不同版本

2. 定向优化
   - 受众精准度
   - 相似受众扩展
   - 排除已转化用户

3. 出价优化
   - CPC vs CPM
   - 自动出价 vs 手动出价
   - 目标ROAS

4. 落地页优化
   - 加载速度
   - 内容相关性
   - 转化流程简化
   - A/B测试

优化优先级：创意 > 定向 > 出价 > 落地页
（按影响程度排序）
```

## 付费引擎适用场景

```
适合付费引擎的产品：

1. 高LTV产品
   - LTV > 3 × CAC
   - 例：SaaS、金融、游戏

2. 目标用户明确
   - 能精准定向
   - 例：企业服务

3. 转化路径清晰
   - 可以追踪转化
   - 例：电商、注册

4. 竞争壁垒
   - 品牌/产品优势
   - 难以被价格战打败

付费引擎的风险：
- 依赖持续投入
- 成本会越来越高
- 竞争激烈时ROI下降
- 没有复利效应

理想策略：
Paid + Viral/Content 组合
付费获取用户 + 有机传播降低CAC
```
