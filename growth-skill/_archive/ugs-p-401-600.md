# Ø 2.精细型ROI计算：

$$
R O I = \frac {L T V}{C A C}
$$

* CAC即平均获客成本 （Customer Acquisition Cost）

# 实例数据

在某自营电商平台上进行广告推广，通过本次推广，带来的新增用户为1000人，15日内产生的总收入为450000元。

# . ROI计算

Step1：计算LTV。

a) 推广费用：总共花费广告费、推广服务费一共68000元；

b) PC成本：由于是贸易型业务，这里的PC成本主要指的是进货成本，该1000人分别购买了电商店铺中的A、B、C三种商品，数量和进货成本分别为A（100件，350元/件），B（160件，150元/件），C（120件，90元/件），则总的进货成本=100*350+160*150+120*90=69800元；

c) 直接人工：主要是1名店长，按30天工资核算，为11000元；推广人员按30天工资核算，为7000元；进货发货人员按40天工资核算，为8000元。则总的直接人工=11000+7000+8000=26000元。

d) 其它：无

总的成本=68000+69800+26000=163800

元

总LTV（净）=450000-163800=286200元

平均LTV=286200/1000=286.2元/人

# 何谓欲望

欲望（Desire）指的是建立在不同的社会经济、文化和个性等基础之上的需要。

# Ø 欲望与兴趣的关系：

• 兴趣反映的状态是“我喜欢”，而欲望反映的状态是“我想要”；

• 有兴趣，不一定会产生欲望；但反过来，如果产生了欲望，通常都会伴随着兴趣的产生。

# Ø 项目背景：

Gorilla 76是一家B2B工业品营销机构，他们在网站上提供金属管材等大宗商品的专业资讯，获得企业客户线索后逐步开展销售转化。

![image](https://cdn-mineru.openxlab.org.cn/result/2026-04-26/18a42c04-acc4-46d4-bf35-918b4d45fd81/c3de43735b41e6798b8fd38be2808a5ed5fd9f91ed0e690711fca8a6d4e3875e.jpg)


# Ø 机会识别：

异常现象：鼠标点击集中于左上角、右下角，但此处并无操作按钮

可能原因：由于这些专业资讯很实用，很多来访者会将信息复制下来保存备用。在复制过程中，鼠标在左上角、右下角形成2次点击

# Ø 制定策略：

• 假设：如果给用户提供更好的资讯保存体验，那么会提高用户获取资料的动机，从而让公司也获得更多的获客线索

优化思路：资料旁边添加一个下载PDF文件的按钮（避免复制下来后产生排版错乱），同时，在PDF页脚位置添加了联系方式

# Ø 验证策略：

• 测试：使总的潜在客户增加了36％，总转化率提高到2.47％

Ø 验证策略：
