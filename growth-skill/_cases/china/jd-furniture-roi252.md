# 京东家具双11投放案例：ROI达到252

## 基本信息

| 字段 | 内容 |
|------|------|
| 案例名称 | 京东家具双11投放案例 |
| 客户 | 艾理思家具京东官方旗舰店（KA店） |
| 产品 | 美式实木家具（沙发、茶几、床柜等） |
| 投放平台 | 京东 |
| 投放时间 | 双11（11月11日） |
| 总预算 | 10万元 |
| 最高ROI | 252 |
| 核心结果 | 双11当天转化位于京东家具TOP1 |

## 完整背景

艾理思家具是一家专注于美式实木家具的电商品牌，在京东平台开设了官方旗舰店。店铺定位中高端市场，主打进口实木和真皮材质的产品，满足追求品质生活的消费者需求。

2020年双11期间，艾理思家具希望借助大促流量实现销售爆发，决定加大推广投放力度。团队通过精细化的投放策略、深入的竞品分析和科学的AIDA转化模型，最终实现了ROI 252的惊人成绩，远超行业平均水平。

本次投放案例不仅展示了家具品类在京东平台的投放策略，更提供了一套可复制的电商投放方法论。

| 挑战 | 说明 |

### 策略5：精细化投放优化

```

## 核心洞察（详细）

### 洞察1：核心价值是转化基础

**产品价值的重要性**：

```
核心价值 vs 促销：

传统思维：
  - 大促=降价
  - 促销=低价
  - 忽略产品价值

艾理思策略：
  - 突出美式风格
  - 强调进口原料
  - 传递时尚品位
  - 价格非唯一因素

效果验证：
  - 创意C：突出核心价值
  - CTR：0.09%（最高）
  - vs 促销创意A：0.04%

结论：
  - 产品核心价值 > 促销低价
  - 价值主张清晰 = 转化率高
  - 用户愿意为价值买单
```

**差异化价值主张**：

```
价值主张构建：

1. 差异化竞争：
   - 美式风格 vs 北欧红海
   - 避免正面竞争
   - 抢占细分市场

2. 原料背书：
   - 进口实木
   - 真皮材质
   - 环保水性漆

3. 情感价值：
   - 高贵情调
   - 品位生活
   - 低调雅致

价值传递：
  - 广告创意：价值优先
  - 着陆页：场景化描述
