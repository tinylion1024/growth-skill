# 标签体系规范

## 标签维度

| 维度 | 说明 | 标准值 |
|------|------|--------|
| **行业** | 所在赛道 | 电商、社交、金融、教育、出海、本地生活、工具、SaaS |
| **打法** | 核心增长策略 | 裂变、会员、内容营销、补贴、算法推荐、订阅、DTC、社区、口碑 |
| **模式** | 商业模式 | 平台、 Marketplace、DTC、订阅、SaaS、免费+付费 |
| **市场** | 地域 | 中国、海外、全球 |
| **阶段** | 企业阶段 | 起步、增长、成熟 |

## 标签示例

```
案例: 拼多多
标签: 电商, 社交裂变, 拼团, 下沉市场

案例: SHEIN
标签: 快时尚, 内容营销, KOC, 出海, DTC

案例: Zoom
标签: SaaS, 工具, 病毒传播, 免费获客

案例: 微信红包
标签: 支付, 社交裂变, 绑定银行卡, 红包

案例: 蚂蚁森林
标签: 金融, 游戏化, 公益, 环保
```

## 使用方式

每个案例建议添加3-5个标签，优先选择最核心的标签。
