## 完整背景

Warby Parker由Neil Blumenthal、Dave Gilboa、Andrew Hunt和Jeffrey Raider于2010年在费城创立。四位创始人相识于沃顿商学院，创业灵感来自Gilboa在购买眼镜时的糟糕体验——一副眼镜要价800美元，但实际成本可能只有50美元。

品牌名字来源于两个虚构角色：Warby佩奇（Parker）和Zagg佩奇（Fletcher），这两个名字是创始人童年时代绰号的混合。

Warby Parker的核心理念是"设计师品质眼镜，平价销售"。通过DTC（直面消费者）模式和创新的Home Try-On服务，彻底颠覆了传统眼镜行业的高价现状。

2021年4月，Warby Parker完成D轮融资，估值超过30亿美元，成为美国估值最高的眼镜品牌。公司从线上起家，逐步开设线下门店，形成O2O闭环模式。

|------|------|
| **Luxottrica垄断** | Luxottrica控制80%+眼镜市场，渠道垄断 |
| **高价眼镜** | 传统眼镜价格虚高（$300-$800） |
| **验光配镜复杂** | 验光、配镜、取镜流程繁琐 |
| **消费者惯性** | 消费者习惯传统眼镜店，对新品牌信任度低 |
| **监管要求** | 眼镜需要验光处方，监管限制 |
| **线上线下融合** | 如何打通线上线下体验 |
