# 模块1：用户增长概述

## 1.1 UGS项目前言

用户增长（User Growth）是企业持续发展的核心动力，核心关注两个维度：
- **用户量增长**：通过各种手段扩大用户规模
- **LTV增长**：提升用户生命周期价值

## 1.2 理解用户增长

### 增长的核心公式

```
增长 = 获客 + 留存 + 变现
```

### 增长的本质

企业增长的目的：**持续以最小成本创造最大价值**

## 1.3 用户增长的衡量

### 用户量指标

| 指标 | 说明 |
|------|------|
| 新增用户数 | 某时间段新增注册用户 |
| 累计用户数 | 总用户池 |
| 获客成本(CAC) | 获取单个用户的成本 |
| 渠道转化率 | 各渠道的转化效率 |

### LTV指标

| 指标 | 说明 |
|------|------|
| LTV | Life Time Value，用户生命周期价值 |
| ARPU | Average Revenue Per User，用户平均收入 |
| 复购率 | 重复购买的比例 |
| 流失率 | 用户流失的比例 |

### 增长健康度

```
LTV > CAC：可持续增长
LTV = 3×CAC：健康线
```

---

## 本模块要点

- 用户增长核心关注「用户量」和「LTV」两个维度
- 增长本质是持续以最小成本创造最大价值
- 衡量体系需同时关注获客效率和用户价值
