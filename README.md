# Growth-Skill 用户增长知识库

> 52个经典增长案例 | 系统方法论 | 实战指南

---

## 项目简介

Growth-Skill 是一个专注于用户增长领域的知识库，涵盖52个经典增长案例、系统方法论和实战指南。

整理了国内外顶级互联网公司的增长实践，帮助你构建完整的增长知识体系。

## 案例结构

每个案例包含9个章节：

| 章节 | 说明 |
|------|------|
| 完整背景 | 公司/产品/行业/阶段 |
| 核心挑战 | 主要增长问题 |
| 增长策略 | 具体方法与执行 |
| 关键数据 | 核心指标与成果 |
| 核心洞察 | 深层规律总结 |
| 可复制点 | 可落地框架 |
| 常见误区 | 需避免的错误 |
| 关键成功因素 | 核心成功要素 |
| 后续演进 | 发展与未来方向 |

## 学习路径

1. **入门** → 阅读 `modules/01-overview.md` 了解增长框架
2. **进阶** → 学习 `guides/` 中的核心模型（AIDA、增长飞轮等）
3. **实战** → 通过 `cases/` 学习具体策略，按标签筛选
4. **复盘** → 应用到自己业务
