## 项目简介

UGS-P（User Growth System Professional）是一个专注于用户增长领域的知识库，涵盖50+经典增长案例、系统方法论和实战指南。

本知识库整理了国内外顶级互联网公司的增长实践，包括电商、社交、金融科技、出海等多个领域，帮助你构建完整的增长知识体系。

## 案例结构

每个案例均包含以下9个章节：

| 章节 | 说明 |
|------|------|
| **完整背景** | 公司/产品/行业/发展阶段 |

| **关键数据** | 核心指标与成果 |

| **可复制点** | 可落地的执行框架 |

## 方法论体系

### 核心模型

- **AIDA模型**：注意力→兴趣→欲望→行动
- **增长飞轮**：自驱动的增长循环
- **北极星指标**：唯一核心指标
- **用户画像**：精准用户定义

### 增长策略

- **获客**：内容营销、SEO、社交广告、病毒传播
- **激活**：新用户引导、首次体验优化
- **留存**：会员体系、Push策略、社区运营
- **变现**：定价策略、付费转化、ARPU提升

### 数据驱动

- **数据分析**：漏斗分析、 cohort分析、归因模型
- **A/B测试**：实验设计、统计显著性
- **用户研究**：调研方法、可用性测试

## 如何使用

### 学习路径

1. **入门**：先阅读 `modules/module-01-overview.md` 了解增长框架
2. **进阶**：学习 `guides/` 中的核心模型和方法论
3. **实战**：通过 `cases/` 中的案例学习具体策略
4. **复盘**：将方法论应用到实际工作

### 案例学习建议

1. **选择相关案例**：找到与你业务最相关的案例
2. **理解背景**：了解当时的市场环境和挑战
3. **拆解策略**：分析具体增长策略和执行细节
4. **提取框架**：总结可复制的增长框架
5. **应用到实践**：结合实际情况落地执行
