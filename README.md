## 项目简介

Growth-Skill 是一个专注于用户增长领域的知识库，涵盖50+经典增长案例、系统方法论和实战指南。

本知识库整理了国内外顶级互联网公司的增长实践，包括电商、社交、金融科技、出海等多个领域，帮助你构建完整的增长知识体系。

## 案例结构

每个案例均包含9个章节：

| 章节 | 说明 |
|------|------|
| **完整背景** | 公司/产品/行业/阶段 |
| **核心挑战** | 主要增长问题 |
| **增长策略（详细拆解）** | 具体方法与执行 |
| **关键数据** | 核心指标与成果 |
| **核心洞察（详细）** | 深层规律总结 |
| **可复制点** | 可落地框架 |
| **常见误区** | 需避免的错误 |
| **关键成功因素** | 核心成功要素 |
| **后续演进** | 发展与未来方向 |

## 方法论体系

### 核心模型
- **AIDA模型**：注意力→兴趣→欲望→行动
- **增长飞轮**：自驱动增长循环
- **北极星指标**：唯一核心指标
- **用户画像**：精准用户定义

### 增长策略
- **获客**：内容营销、SEO、社交广告、病毒传播
- **激活**：新手引导、首单转化
- **留存**：会员体系、Push召回、社区运营
- **变现**：定价策略、付费转化、ARPU提升
