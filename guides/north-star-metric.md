# 北极星指标

## 定义

北极星指标（North Star Metric）是产品成功的唯一关键指标，是团队共同努力的方向。

## 特征

- **清晰可衡量**：能用数据明确表达
- **反映用户价值**：能体现用户获得的价值
- **预测长期增长**：能预示业务的可持续增长
- **全团队理解**：所有成员都能理解其含义

## 为什么重要

- 聚焦团队注意力
- 统一团队目标
- 避免资源浪费
- 衡量产品成功

---

## 拆解方法

| 方法 | 适用场景 | 说明 |
|------|----------|------|
| 按使用场景拆解（小） | 设定零碎工作目标 | 从功能维度拆解 |
| 按OKR思路拆解（中） | 明确项目目标 | 从目标-关键结果拆解 |
| 按用户旅程拆解（大） | 寻找新的项目机会 | 从AARRR各阶段拆解 |

### 按使用场景拆解

适用场景：某产品有多个功能，需要为各功能设定目标

示例：
- 北极星指标：总销售额
- 拆解：搜索功能使用率、购物车使用率、支付成功率

### 按OKR思路拆解

适用场景：季度/年度目标制定

示例：
```
O1: 提升产品转化率
  KR1: 降低BUG数量20%
  KR2: 提升首页转化10%

O2: 增加商户及产品数量
  KR3: 上线商品推荐功能
```

### 按用户旅程拆解

适用场景：寻找新的增长机会

示例：
```
北极星指标：总销售额

用户旅程：
访问网站 → 注册账户 → 第一次购买 → 重复购买

增长指标：
新用户访问量 × 注册率 × 首次购买率 × 客单价 × 复购率
```

---

## 拆解工具：用户增长地图

```
北极星指标
    ↓
用户旅程各阶段
    ↓
各阶段增长指标
    ↓
二级洞察和机会
    ↓
后台支持需求
```

---

## 案例：某电商产品

**北极星指标**：总销售额

**用户旅程拆解**：
- 访问电商网站
- 注册账户
- 第一次购买
- 重复购买

**增长指标**：
| 阶段 | 增长指标 |
|------|----------|
| 获取 | 新用户访问量 |
| 激活 | 用户注册率 |
| 留存 | 首次购买率 |
| 变现 | 新用户平均订单金额 |
| 推荐 | 老用户复购率、老用户平均订单金额 |

**二级洞察**：
- 用户关注... → 需要差异化宣传方式
- 注册能得到... → 需要差异化价值体现
- 用户付费动力是... → 需要差异化引导

## 常见北极星指标示例

| 行业 | 北极星指标示例 |
|------|---------------|
| 社交 | 日活用户数 |
| 电商 | GMV |
| 内容 | 总阅读时长 |
| 游戏 | 日活用户数 |
| SaaS | 付费转化率 |
| 金融 | 管理资产规模 |

---

## 注意事项

1. **唯一性**：一个产品通常只有一个北极星指标
2. **阶段性**：不同阶段可能有不同的北极星指标
3. **可执行**：必须是团队可以影响的指标
4. **可分解**：能够拆解为可执行的子指标

---

## 小结

- 北极星指标是增长的核心指引
- 三种拆解方法适配不同场景
- 用户旅程拆解适合寻找新机会
- 拆解后需配合洞察和假设验证
