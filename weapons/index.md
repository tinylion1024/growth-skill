## 🎯 使用指南

### 按阶段选择

```
前期（0-1）：
  冷启动 + 病毒裂变 + 内容增长 + 社区增长

中期（1-10）：
  PLG + 留存 + 变现 + 付费广告

后期（10-100）：
  品牌 + B2B销售 + 规模化
```

### 按产品选择

```
To C 产品：
  病毒裂变 > 社区增长 > 内容增长 > PLG

To B 产品：
  内容增长 > B2B销售 > PLG > 品牌

Marketplace：
  病毒裂变 > 社区增长 > PLG > 变现

SaaS：
  PLG > 留存 > 变现 > 内容增长
```
