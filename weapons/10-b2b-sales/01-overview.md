# B2B销售增长武器库

> 企业客户的8个增长武器

---

## 概述

B2B销售增长是通过销售团队获取企业客户。核心是建立高效的B2B销售流程和团队。

```
B2B增长公式：

B2B Growth = Leads × Conversion × Deal Size × Cycle

核心逻辑：
精准线索 → 专业转化 → 大额成交 → 长期复购
```

---

## 武器清单（8个）

### 104. 冷启动外呼

```
战术：通过电话开发企业客户

外呼流程：

准备阶段：
- 目标客户画像
- 联系方式获取
- 话术准备

开场白：
- 自我介绍
- 价值钩子
- 约时间

话术模板：
"我是X公司的Y，我们帮助Z企业解决什么问题。您现在方便吗？"

跟进策略：
- 多轮触达
- 邮件+电话组合
- LinkedIn追加

关键指标：
- 接通率
- 转化到Demo
- 平均周期

工具：
- 销售加速平台
- 线索数据库
- 通话分析
```

### 105. LinkedIn外联

```
战术：通过LinkedIn触达企业决策者

外联策略：

个人资料优化：
- 专业头像
- 清晰标题
- 价值描述

连接请求：
- 个性化
- 价值说明
- 简短清晰

后续消息：
- 提供价值
- 明确目的
- 约时间

内容策略：
- 行业洞察
- 专业分享
- 建立信任

案例：
- 创始人IP
- 客户故事
- 数据报告

关键指标：
- 连接通过率
- 响应率
- 转化率
```

### 106. Webinar

```
战术：通过在线研讨会获取销售线索

策划流程：

主题选择：
- 痛点相关
- 解决方向
- 案例分享

内容设计：
- 开场（价值承诺）
- 内容（干货）
- 案例（证明）
- CTA（下一步）

执行要点：
- 提前宣传
- 互动设计
- 后续跟进

工具：
- Zoom Webinar
- Demio
- Hopin

关键指标：
- 注册率
- 出席率
- 线索转化率
- 出席后转化率
```

### 107. Demo演示

```
战术：通过产品演示促进销售决策

演示类型：

On-Demand Demo（录播）：
- 标准内容
- 自助观看
- 自动化跟进

Live Demo（直播）：
- 定制化
- 实时互动
- 深度了解

演示准备：
- 了解客户痛点
- 准备相关案例
- 演示环境测试

演示结构：
1. 了解情况（5分钟）
2. 解决方案（20分钟）
3. 案例证明（10分钟）
4. 后续路径（5分钟）

关键指标：
- Demo完成率
- Demo后转化率
- 平均成单周期
```

### 108. 白皮书下载

```
战术：通过专业内容获取销售线索

白皮书类型：
- 行业报告
- 方法论
- 技术白皮书
- 最佳实践

落地页设计：
- 标题吸引
- 内容介绍
- 表单收集

后续跟进：
- 自动邮件序列
- 销售电话
- 个性化内容

分发渠道：
- 官网
- 邮件营销
- 付费推广
- 合作伙伴

关键指标：
- 下载量
- 线索转化率
- MQL到SQL转化
```

### 109. 销售自动化

```
战术：用工具提升销售效率

自动化场景：

线索管理：
- 线索评分
- 线索分配
- 自动录入

跟进自动化：
- 邮件序列
- 任务提醒
- 电话提醒

报告自动化：
- 活动报告
- 管道报告
- 预测报告

工具：
- HubSpot Sales
- Salesforce
- Outreach
- Salesloft

关键点：
- 不要过度自动化
- 保持人性化
- 持续优化
```

### 110. 客户成功

```
战术：通过客户成功团队提升续费

职责：
- 客户 onboarding
- 使用培训
- 价值实现
- 续费管理

关键指标：
- 首次价值实现时间
- 客户满意度
- 续费率
- 净推荐值

服务模式：
- 主动服务（定期沟通）
- 被动服务（响应需求）
- 混合模式

工具：
- 客户健康度仪表盘
- 自动化提醒
- 定期检查

关键点：
- 以客户价值为导向
- 提前预警问题
- 建立长期关系
```

### 111. 转介绍计划

```
战术：通过现有客户推荐新客户

推荐激励：
- 现金奖励
- 服务延期
- 荣誉认可
- 升级服务

触发时机：
- 项目交付满意
- 续费完成
- 问题解决后

流程：
- 询问机会
- 提供素材
- 跟踪感谢

工具：
- Referral奖励系统
- 推荐追踪
- 自动化感谢

关键指标：
- 推荐率
- 推荐转化率
- 推荐客户LTV

关键点：
- 时机很重要
- 要提供方便
- 感谢要及时
```

## B2B销售检查清单

```
B2B销售评估：

□ 线索数量和质量？
□ 转化率？
□ 平均deal size？
□ 销售周期？

销售效率：
- 销售流程是否清晰？
- 工具是否高效？
- 团队是否专业？
```
