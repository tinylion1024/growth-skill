# 付费广告增长武器库

> 效果营销的10个增长武器

---

## 概述

付费广告增长是通过投放广告获取用户。核心是建立正向ROI的获客渠道。

```
付费增长公式：

Paid Growth = Ad Spend × ROAS

核心逻辑：
投入广告 → 获取用户 → 产生收入 → 收入 > 成本
```

---

## 武器清单（10个）

### 86. Google Ads

```
战术：搜索广告获取主动需求用户

广告类型：
- 搜索广告（Search）
- 显示广告（Display）
- 购物广告（Shopping）

关键词策略：
- 品牌词（保护）
- 行业词（获客）
- 长尾词（精准）

出价策略：
- CPC（按点击）
- CPA（按转化）
- ROAS（目标回报）

账户结构：
- 广告系列
- 广告组
- 关键词
- 广告

关键指标：
- CTR（点击率）
- CVR（转化率）
- CPA（单次获取成本）
- ROAS（广告回报）

优化方向：
- 关键词质量分
- 广告相关度
- 落地页体验
```

### 87. Facebook Ads

```
战术：社交广告精准触达目标用户

广告类型：
- 图文广告
- 视频广告
- 轮播广告
- 集合广告

定向能力：
- 人口统计（年龄/性别）
- 兴趣定向
- 行为定向
- 自定义受众

定向策略：
- Lookalike相似人群
- 再营销受众
- 兴趣层叠

创意要点：
- 前3秒抓住注意力
- 清晰的CTA
- 社会证明

关键指标：
- CPM（千次展示成本）
- CPC（单次点击成本）
- CVR（转化率）
- CPA（单次获取成本）
```

### 88. TikTok Ads

```
战术：短视频平台获取年轻用户

广告类型：
- 信息流广告
- 开屏广告
- 挑战赛广告

内容特点：
- 原生感强
- 竖屏为主
- 前3秒关键

定向能力：
- 兴趣定向
- 行为定向
- 年龄/性别

创意策略：
- 真人出镜
- 展示真实使用
- 有趣/有共鸣

关键指标：
- 视频完播率
- 互动率
- 转化率
- CPA
```

### 89. YouTube Ads

```
战术：视频广告建立品牌和转化

广告类型：
- 可跳过的插播广告（Skip Ad）
- 不可跳过的插播广告
- 发现广告
- 短视频广告（Shorts）

策略：
- 品牌曝光+搜索配合
- 教程内容+产品转化
- 案例故事+信任建立

制作要点：
- 前5秒要抓住
- 内容要有价值
- CTA要清晰

关键指标：
- 观看率
- 完播率
- 转化率
- 每次观看成本
```

### 90. 再营销广告（Retargeting）

```
战术：触达已访问但未转化的用户

使用场景：
- 访问未注册
- 注册未试用
- 试用未付费
- 付费未续费

平台：
- Google Retargeting
- Facebook Retargeting
- TikTok Retargeting

策略：
- 新用户：价值教育
- 活跃用户：激活转化
- 流失用户：召回

创意要点：
- 个性化（访问页面）
- 明确的价值
- 限时优惠

关键指标：
- 转化率（高于新用户）
- CPA（低于新用户）
- ROAS
```

### 91. Lookalike人群

```
战术：基于现有客户拓展相似新用户

创建方式：
- 基于购买客户
- 基于高价值用户
- 基于活跃用户

相似度选择：
- 1%：最相似（高质量）
- 5%：中等相似（规模）
- 10%：较宽泛（规模优先）

使用策略：
- 先用高相似度测试
- 效果好后扩展
- 多层Lookalike叠加

关键指标：
- 相似度质量
- 转化率
- CPA vs 现有用户
```

### 92. App Store Ads

```
战术：在应用商店内投放获取用户

平台：
- Apple Search Ads
- Google Play Ads

关键词策略：
- 相关度高
- 竞争度适中
- 搜索量合适

优化要点：
- 截图和视频
- 标题和描述
- 评分和评论

出价策略：
- CPA优化
- 关键词竞价

关键指标：
- 安装率
- CPA
- 留存率（质量）
```

### 93. KOL投放

```
战术：通过关键意见领袖推广

合作方式：
- 代言推广
- 产品植入
- 测评体验
- 直播带货

KOL类型：
- 头部KOL（高预算）
- 腰部KOL（性价比）
- 素人KOC（口碑）

选择标准：
- 受众匹配
- 互动率
- 内容质量
- 合作历史

关键指标：
- 曝光量
- 互动率
- 转化率
- ROI
```

### 94. 联盟营销（Affiliate）

```
战术：通过联盟伙伴分销获客

联盟类型：
- 个人推广者
- 媒体/博客
- 对比网站
- 返利网站

佣金设计：
- 按销售分成（CPS）
- 按引导付费（CPL）
- 按点击付费（CPC）

管理平台：
- Impact
- ShareASale
- 独立系统

关键指标：
- 联盟伙伴数量
- 订单转化率
- 佣金支出
- ROI
```

### 95. Influencer合作

```
战术：与影响者合作推广

合作形式：
- 贴片广告
- 内容合作
- 挑战赛
- 带货分成

选择标准：
- 受众匹配度
- 真实影响力
- 内容调性
- 合作配合度

执行要点：
- 明确目标
- 内容创意自由度
- 效果追踪
- 长期关系

关键指标：
- 曝光量
- 互动成本
- 转化率
- 品牌提升
```
