# 游戏化增长

> 积分 + 任务 + 成就 = 习惯养成 + 用户粘性

---

## 核心理论

**游戏化增长 = 用游戏机制让产品变得"上瘾"**

```
游戏化的本质：
- 让用户做你想让他做的事
- 让这个过程充满乐趣
- 让用户持续回来

关键区别：
非游戏化：功能驱动（你必须用）
游戏化：内在动机（你想用）

内在动机三要素：
1. 成就感（完成挑战）
2. 归属感（加入社群）
3. 自主感（选择自由）
```

## 关键框架

### 游戏化设计框架（八角分析）

```
┌─────────────────────────────────────────┐
│                                         │
│         ⬆ 史诗级意义感                  │
│        ↗           ↖                    │
│  ↗                 ↖                   │
│ ⬅   收集/发现    授权/创意  ➡         │
│  ↙                 ↘                   │
│  ↘                 ↙                   │
│ ⬅   社交/竞争    回避/惩罚  ➡         │
│  ↖                 ↙                   │
│        ↙           ↘                   │
│         ⬇ 紧迫感                        │
│                                         │
└─────────────────────────────────────────┘

核心驱动力（左侧）：
- 史诗级意义感（使命感）
- 社交属性（竞争/合作）
- 收集（收集所有勋章）
- 探索（发现新内容）

外在驱动力（右侧）：
- 回避惩罚（连续签到惩罚）
- 限时优惠（倒计时）
- 授权（等级解锁）
- 创意（自定义角色）
```

### 用户行为循环

```
触发（Trigger）
    │
    ▼
行动（Action）
    │
    ▼
奖励（Reward）
    │
    ├── 即时奖励（多巴胺）
    │       │
    │       ▼
    │   可变奖励（Variable Reward）
    │       │
    │       ▼
    │   投入（Investment）
    │       │
    │       ▼
    └── 下一轮触发 ←────┘
```

### 积分体系设计

```
┌────────────────────────────────────────────┐
│  积分获取                                  │
│                                            │
│  基础行为（每日活跃）                      │
│  - 每日登录：+5                            │
│  - 分享内容：+10                           │
│                                            │
│  核心行为（产品价值）                      │
│  - 完成首次购买：+100                      │
│  - 推荐好友：+50                            │
│  - 发布UGC：+30                            │
│                                            │
│  社交行为（传播增长）                      │
│  - 邀请注册：+80                           │
│  - 好友互动：+15                           │
└────────────────────────────────────────────┘

积分消耗：
- 兑换礼品（物质激励）
- 解锁功能（精神激励）
- 等级提升（身份象征）
```

## 游戏化设计检查清单

```
□ 核心行为明确（你想让用户做什么）
□ 奖励机制设计（即时+可变+长期）
□ 进度可视化（等级/积分/进度条）
□ 社交元素（竞争/合作/分享）
□ 触发机制（Push/邮件/红点）
□ 惩罚机制（连续签到/积分清零）
□ 测试迭代（A/B测试不同方案）
□ 避免过度（不损害产品体验）
```
