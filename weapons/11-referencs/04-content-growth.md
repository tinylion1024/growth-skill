# 内容增长

> SEO + 内容矩阵 + 自媒体 = 持续免费流量

---

## 核心理论

**内容增长 = 用有价值的内容持续获取免费流量**

```
内容增长的核心逻辑：

创作内容 → 搜索引擎收录 → 持续获得流量 → 用户转化

关键特点：
- 长期有效（一次创作，长期收益）
- 成本递减（优质内容越老越有价值）
- 品牌积累（内容即品牌资产）
- 流量稳定（不依赖付费广告）
```

**内容增长 vs 付费广告**：

| 维度 | 内容增长 | 付费广告 |
|------|---------|---------|
| 见效周期 | 3-6个月 | 立即 |
| 持续时间 | 长期 | 停止即停 |
| 边际成本 | 递减 | 递增 |
| 品牌价值 | 高 | 低 |
| 可规模化 | 高 | 中 |

```
              └────────────────┘
```

### SEO 关键词策略

```
关键词类型：

1. 导航类（Navigational）
   搜索品牌名，寻找官网
   价值低，不推荐主攻

2. 信息类（Informational）
   "如何..." "是什么..."
   教育用户，建立信任

3. 商业类（Commercial）
   "best [product]"
   比较研究，高意向

4. 交易类（Transactional）
   "buy [product]"
   高转化，但竞争激烈

策略：
- 新品 → 先攻信息类
- 成长期 → 攻商业类
- 成熟期 → 攻交易类
```

### 内容复用矩阵

```
一次创作，多平台分发：

原始内容（博客文章）
     │
     ├── 视频脚本 → YouTube / TikTok
     ├── 音频 → Podcast
     ├── 摘要 → Twitter / LinkedIn
     ├── 系列 → Newsletter
     ├── 幻灯片 → SlideShare
     └── 问答 → 播客节目 / 论坛
```
