# 付费广告增长

> ROI投放 + 人群定向 = 可控的规模化增长

---

## 核心理论

**付费广告增长 = 用钱买时间，快速获取目标用户**

```
付费广告的核心逻辑：
- 付费 = 立即获得流量
- 停止付费 = 立即停止流量
- 优化目标：让每一分钱带来最大的回报

与传统增长的对比：
内容增长：慢启动，长期有效，边际成本递减
付费广告：快启动，短期有效，边际成本递增

最佳策略：两者结合
- 付费广告快速验证
- 内容增长承接流量
- 数据驱动持续优化
```

## 关键框架

### 广告投放漏斗

```
         Top（认知）
            │
            ▼
┌───────────────────────────┐
│  曝光 → 点击              │  漏斗顶部
│  Impression → Click       │  量大，成本低
└───────────────────────────┘
            │
            ▼
┌───────────────────────────┐
│  点击 → 访问              │  漏斗中部
│  Click → Visit            │  优化落地页
└───────────────────────────┘
            │
            ▼
┌───────────────────────────┐
│  访问 → 转化              │  漏斗底部
│  Visit → Conversion       │  精准，高价值
└───────────────────────────┘
```

### 核心指标

```
广告效果衡量指标：

CPM（千次曝光成本）
  = 花费 / 曝光量 × 1000
  适用于品牌曝光类广告

CPC（单次点击成本）
  = 花费 / 点击量
  适用于引流类广告

CPA（单次行动成本）
  = 花费 / 转化数
  适用于转化导向广告

LTV（用户生命周期价值）
  = 用户带来的总收入
  判断是否值得投放的依据

ROI（投资回报率）
  = (收入 - 花费) / 花费
  > 0 才算盈利

CAC（用户获取成本）
  = 花费 / 获取用户数
  需与LTV对比才有意义
```

### 人群定向策略

```
基础定向：
- 地域 / 年龄 / 性别
- 兴趣 / 行为

高级定向：
- 再营销（Visitores）
- Lookalike（相似人群）
- Custom Audience（自定义人群）
- LAL（相似扩展）

定向原则：
- 先窄后宽（先测试再扩展）
- 精准 > 广泛（ROI更高）
- 数据驱动（持续优化）
```

## 付费广告检查清单

```
□ 目标明确（品牌/引流/转化）
□ 受众精准（先窄后宽）
□ 素材测试（A/B测试至少3组）
□ 落地页优化（与广告承诺一致）
□ 数据追踪（UTM参数完整）
□ ROI计算（LTV > CAC）
□ 持续优化（每日/每周复盘）
□ 渠道组合（不依赖单一渠道）
```
