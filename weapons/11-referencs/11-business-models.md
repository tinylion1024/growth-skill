# 商业模型

> LTV/CAC + 指标驱动 = 可持续的商业增长

---

## 核心理论

**商业模型增长 = 确保增长的同时商业模式成立**

```
增长 ≠ 商业成功

增长黑客关注：用户增长
商业模型关注：盈利能力

核心公式：
单位经济模型 = LTV > CAC × 3（健康线）

LTV（用户生命周期价值）
  = 用户带来的总收入
  = ARPU × 留存时长

CAC（用户获取成本）
  = 获取一个用户花的钱

LTV/CAC > 3：商业模式健康
LTV/CAC < 1：商业模式危险
```

---

## 关键框架

### 核心商业指标

```
收入相关：
- ARPU（每用户平均收入）
- LTV（用户生命周期价值）
- 付费转化率
- 客单价

成本相关：
- CAC（用户获取成本）
- CRT（客户保留成本）
- COGS（销售成本）

健康指标：
- LTV/CAC（>3 健康）
- Payback Period（<12个月）
- NPS（>50 良好）

增长指标：
- MRR（每月经常性收入）
- Churn Rate（月流失率）
- NRR（净收入留存率）
```

### 订阅经济模型

```
月度订阅：
收入 = 用户数 × 月费
增长驱动：新增 - 流失

年度订阅：
收入 = 用户数 × 年费
优势：提前锁定收入，降低流失

关键指标：
- MRR（月度经常性收入）
- ARR（年度经常性收入）
- Churn（流失率）
- NRR（净收入留存）

增长策略：
- 降低流失（留存）
- 提升ARPU（向上销售）
- 扩展收入（Cross-sell）
```

### 单位经济模型

```
计算示例（SaaS）：

CAC = $100（获取一个用户花100美元）
月费 = $30
毛利率 = 80%
月流失率 = 5%

LTV = 月费 × 毛利率 × 平均生命周期
    = $30 × 80% × (1/5%)
    = $30 × 80% × 20个月
    = $480

LTV/CAC = $480 / $100 = 4.8 > 3 ✓ 健康

Payback Period = CAC / (月费 × 毛利率)
               = $100 / ($30 × 80%)
               = 4.2个月 ✓ 健康
```

## 商业模型检查清单

```
□ LTV/CAC > 3（单位经济健康）
□ Payback Period < 12个月
□ 流失率可控（月流失 < 5%）
□ 定价策略清晰
□ 成本结构透明
□ 财务模型可预测
□ 有明确的盈利路径
□ 规模化的边际成本递减
```
