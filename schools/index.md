## 🎯 流派选择指南

### 按产品阶段

```
冷启动阶段（0-1）：
  主要流派：内容增长 > 社区增长 > 病毒传播
  原因：低成本获取种子用户

产品市场匹配（1-10）：
  主要流派：增长黑客 > 产品驱动 > 病毒传播
  原因：验证产品价值和增长假设

规模化增长（10-100）：
  主要流派：网络效应 > 付费广告 > 内容增长
  原因：高效获取大量用户

成熟稳定期（100+）：
  主要流派：品牌驱动 > 增长飞轮
  原因：建立护城河，长期健康增长
```

### 按产品类型

```
To C 产品（消费者）：
  必学：病毒传播、游戏化、品牌驱动
  可选：增长黑客、付费广告

To B 产品（企业）：
  必学：内容增长、产品驱动、社区增长
  可选：网络效应、付费广告

Marketplace（平台）：
  必学：网络效应、增长黑客
  可选：病毒传播、社区增长

SaaS（软件）：
  必学：产品驱动、增长黑客
  可选：付费广告、社区增长
```

## 📦 增长团队构成

大公司增长团队通常对应不同流派：

```
增长团队配置：

├─ 增长营销团队
│   ├─ 付费广告（Performance Marketing）
│   ├─ 内容增长（Content Marketing）
│   └─ SEO
│
├─ 增长产品团队
│   ├─ 产品驱动增长（PLG）
│   ├─ 病毒传播（Viral）
│   └─ 游戏化（Gamification）
│
├─ 增长运营团队
│   ├─ 社区运营（Community）
│   ├─ 裂变运营（Referral）
│   └─ 用户运营（Retention）
│
└─ 品牌团队
    ├─ 品牌策略（Brand）
    └─ 品牌传播（Campaign）
```
