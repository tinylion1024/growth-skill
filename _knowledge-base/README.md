# UGS-P 用户增长体系 知识库

> 用户增长运营师（UGS-P）认证课程知识整理

## 课程概述

UGS-P（User Growth Specialist - Professional）是SMEI国际营销协会推出的专业认证体系，聚焦用户增长的核心能力：
- **用户量增长**：通过获客、裂变等方式扩大用户规模
- **LTV增长**：通过留存、深耕提升用户生命周期价值

## 核心术语表

| 术语 | 释义 |
|------|------|
| **LTV** | Life Time Value，用户生命周期价值 |
| **北极星指标** | North Star Metric，核心增长指标 |
| **AIDA** | Attention → Interest → Desire → Action 转化模型 |
| **PDB** | Programmatic Direct Buy，程序化直接购买 |
| **PD** | Preferred Deal，优先交易 |
| **PA** | Private Auction，私有竞价 |
| **RTB** | Real-Time Bidding，实时竞价 |
| **RFM** | Recency, Frequency, Monetary，会员价值模型 |
| **AARRR** | Acquisition → Activation → Retention → Revenue → Referral |
| **PMF** | Product-Market Fit，产品市场契合度 |
| **魔法数字** | Magic Number，促进用户激活的关键行为次数 |
| **五度模型** | 吸引度、完成度、满意度、忠诚度、推荐度 |
