## 🎯 流派选择指南

### 按产品阶段

```
冷启动（0-1）：
  增长黑客 + 内容增长 + 社区增长

产品市场匹配（1-10）：
  产品驱动增长 + 病毒传播 + 网络效应

规模化（10-100）：
  付费广告 + 网络效应 + 内容增长

成熟稳定（100+）：
  品牌驱动 + 增长飞轮 + 商业模型
```

### 按产品类型

```
To C 产品：病毒传播 > 游戏化 > 品牌驱动
To B 产品：内容增长 > 商业模型 > B2B销售
Marketplace：网络效应 > 社区增长 > 产品驱动
SaaS：产品驱动 > 增长飞轮 > 商业模型
```

## 🧭 学习路径

```
入门（1-2月）：
  增长黑客派 → 产品驱动增长 → 病毒传播

进阶（3-4月）：
  网络效应 → 内容增长 → 社区增长

高阶（5-6月）：
  品牌驱动 → 增长飞轮 → 商业模型
```
