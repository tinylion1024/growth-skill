## SEO基础

### 关键词研究

```
关键词研究流程：

Step 1: 发现关键词
   - 用户在搜索什么
   - 竞品在做什么
   - 行业趋势是什么

Step 2: 分析关键词
   - 搜索量（Search Volume）
   - 竞争度（Competition）
   - 商业价值（Commercial Value）

Step 3: 选择关键词
   - 高搜索量 + 低竞争度 = 机会
   - 长尾关键词 = 精准流量

Step 4: 内容布局
   - 首页：核心关键词
   - 栏目页：中长尾
   - 内容页：长尾词
```

### 内容优化

```
On-Page SEO清单：

标题优化：
- 包含目标关键词
- 吸引点击（数字、情绪）
- 长度控制在60字符内

内容结构：
- H1：页面主题
- H2：主要章节
- H3：细分观点
- 每段不超过3-4句

技术优化：
- URL包含关键词
- 图片Alt文本
- 页面加载速度
- 移动端友好

内链策略：
- 相关文章内链
- 锚文本优化
- 层级结构清晰
```

## 内容策略

### Skyscraper Technique

```
 skyscraper technique 三步法：

Step 1: 找到优秀内容
   - 搜索行业关键词
   - 找到排名靠前的内容
   - 分析其优点和缺点

Step 2: 做得更好
   - 更全面（添加缺失部分）
   - 更深度（扩展每个观点）
   - 更现代（更新数据）
   - 更好看（优化排版）

Step 3: 主动推广
   - 联系提到原文的网站
   - 联系链接到原文的网站
   - 推广到相关社区

案例：
Brian Dean用这个方法，流量3个月内提升110%
```

## 常见误区

```
内容增长误区：

❌ 追求数量忽视质量
   - 10篇垃圾不如1篇精品
   - 搜索引擎惩罚低质量

❌ 只做SEO不顾用户体验
   - 关键词堆砌
   - 可读性差

❌ 忽视内容分发
   - 内容好但没人看到
   - 分发和内容一样重要

❌ 没有系统化
   - 随心情写内容
   - 没有内容日历

✅ 正确做法：
- 质量优先，系统化生产
- SEO + 用户体验平衡
- 内容 + 分发同步
```
