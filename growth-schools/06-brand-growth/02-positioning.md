```
定位决定增长

传统思维：             定位思维：

功能比对手强           认知>事实
酒香不怕巷子深         位置 > 产品
```

---

## 定位三问

```
定位 = 回答一个问题：

为什么用户应该选择你而不是竞争对手？

定位三问：
1. 你是谁？（品牌名）
2. 你是什么？（品类）
3. 为什么选你？（差异化）
```

---

## 定位六步法

```
Step 1: 分析市场
   - 市场有多大？
   - 主要玩家是谁？
   - 用户需求是什么？

Step 2: 确定竞争对手
   - 用户当前用什么？
   - 替代方案是什么？

Step 3: 选择差异化
   - 你与竞品的不同是什么？
   - 这个不同对用户重要吗？

Step 4: 建立信任状
   - 如何证明你的差异化？
   - 有什么背书？

Step 5: 传播定位
   - 用一句话说明你的定位
   - 一致性传播

Step 6: 持续优化
   - 监控认知变化
   - 随市场调整
```

## Purple Cow 效应

```
Seth Godin 的 Purple Cow 理论：

传统营销：找到目标消费者 → 打断他们 → 强迫看广告

Purple Cow：创造一个值得注意的产品 → 让消费者主动谈论

Remarkable 标准：
1. 必须 remarkable（值得被谈论）
2. 必须可视化（能被看到/分享）
3. 必须可复制（能大规模推广）
4. 必须有意义（对用户有价值）
```

## 品类思维

```
品类 > 产品

用户决策路径：
"我想喝点什么？"
    │
    ├── 碳酸饮料 ── 可口可乐
    ├── 果汁 ────── 农夫山泉
    └── 水 ─────── 农夫山泉

品类选择：
- 创造新品类
- 占领新品类
- 领先新品类
```

## 品牌叙事框架

```
故事 = 冲突 + 转折 + 启示

案例 - Apple：
冲突：电脑只是工具，没有美感
转折：Apple相信科技可以很美
启示：Think Different，改变世界

案例 - Nike：
冲突：运动很辛苦，容易放弃
转折：Nike相信每个人都能成为运动员
启示：Just Do It，突破自我
```
