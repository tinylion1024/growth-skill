# 付费广告增长

> David Ogilvy | ROI驱动的效果营销

---

## 核心理念

```
付费广告增长 = 通过投入广告获取用户

核心公式：
Growth = f(Ad Spend, ROAS, Scale)

核心原则：
1. 每一分钱都要追踪回报
2. ROAS必须为正才能扩张
3. 渠道选择要精准
4. 创意和定向同样重要
```

---

## 付费广告体系

### 主要广告平台

```
主流广告平台：

搜索引擎广告：
- Google Ads（全球最大）
- 百度推广（中国市场）
- 适合：明确需求，用户主动搜索

社交媒体广告：
- Facebook/Instagram Ads
- TikTok Ads
- 适合：品牌曝光，精准定向

展示广告：
- Google Display Network
- 程序化购买
- 适合：品牌曝光，引流网站

视频广告：
- YouTube Ads
- 抖音/快手信息流
- 适合：品牌+转化

电商广告：
- 淘宝直通车
- 亚马逊 Sponsored Ads
- 适合：电商转化
```

### 渠道选择框架

```
渠道选择决策：

Step 1: 确定目标
   - 品牌曝光
   - 转化
   - 品牌+转化

Step 2: 了解用户
   - 目标用户在哪？
   - 他们的搜索行为？
   - 他们的社交行为？

Step 3: 测试渠道
   - 小预算测试
   - 收集数据
   - 优化后扩张

渠道优先级：
1. 目标用户在哪就去哪
2. 能追踪转化的渠道
3. ROAS为正的渠道
4. 能规模化的渠道
```

## 效果营销指标

### 核心指标

```
效果营销核心指标：

基础指标：
- Impressions（展示）
- Clicks（点击）
- CTR（点击率）

转化指标：
- Conversions（转化）
- CVR（转化率）
- CPA（单次获取成本）

收入指标：
- Revenue（收入）
- ROAS（广告支出回报率）
- Profit（利润）

公式：
ROAS = Revenue / Ad Spend
CPA = Ad Spend / Conversions
CVR = Conversions / Clicks
CTR = Clicks / Impressions
```

### ROAS分析

```
ROAS决策框架：

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   ROAS > 5x    优秀   - 大力扩张                               │
│                                                                 │
│   ROAS 3-5x    良好   - 稳定投入                               │
│                                                                 │
│   ROAS 1-3x    一般   - 优化效率                               │
│                                                                 │
│   ROAS < 1x    亏损   - 立即停止                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

ROAS优化方向：
- 提高转化率（优化落地页）
- 提高客单价
- 降低单次点击成本
- 优化用户生命周期价值
```

## 广告优化

### 创意优化

```
广告创意优化：

注意力阶段：
- 前3秒抓住眼球
- 视觉 > 文字
- 有趣/震惊/共鸣

兴趣阶段：
- 明确用户痛点
- 展示解决方案
- 社会证明

行动阶段：
- 清晰的CTA
- 紧迫感/稀缺
- 简化行动步骤

创意测试：
- 同时测试多个版本
- 变量控制
- 统计显著后再决策
```

### 定向优化

```
定向优化策略：

基础定向：
- 人口统计（年龄、性别）
- 地域
- 设备

行为定向：
- 兴趣
- 购买意图
- 自定义受众

相似受众：
- Lookalike Audience
- 基于现有客户拓展

再营销：
- 网站访客
- App用户
- 互动用户

定向优先级：
1. 自定义受众（最精准）
2. 再营销（高意向）
3. 相似受众（规模+精准）
4. 兴趣定向（宽泛）
```

## 电商广告

### 电商投放策略

```
电商广告策略：

上架期（0-30天）：
- 测款为主
- 预算 = 款数 × 测试预算
- 目标：找到潜力款

成长期（30-90天）：
- 集中投放潜力款
- 优化关键词和创意
- 目标：降低ACOS

稳定期（90天+）：
- 稳定投放+持续优化
- 拓展渠道
- 目标：提升GMV

广告类型组合：
- 拉新广告（新品/爆款）
- 防守广告（维护排名）
- 品牌广告（品牌建设）
```

### ACOS分析

```
ACOS（广告成本销售比）：

ACOS = Ad Spend / Revenue × 100%

ACOS 参考：
- < 15%   优秀
- 15-25%  良好
- 25-35%  一般
- > 35%   需要优化

ACOS优化方向：
- 优化关键词（提高转化）
- 优化创意（提高点击）
- 优化落地页（提高转化）
- 优化定价（提高利润）
```

## 常见误区

```
付费广告误区：

❌ 只看CTR
   - 点击高不等于转化高
   - 要看最终ROI

❌ 只看CPA
   - CPA低不等于好
   -要看用户LTV

❌ 忽视创意
   - 定向再好，创意烂也没用
   - 创意和定向同样重要

❌ 过度依赖付费
   - 付费只是增长的一部分
   - 需配合其他增长方式

❌ 不测试就大规模
   - 先小规模测试
   - 统计显著后再扩张

✅ 正确做法：
- 追踪到最终转化
- 平衡短期ROI和长期LTV
- 创意和定向同步优化
- 测试后扩张
```
